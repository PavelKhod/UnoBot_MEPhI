@unobot
https://github.com/jh0ker/mau_mau_bot

Стек: Python 3.4, python-telegram-bot, Pony ORM (работа с БД), bash (вспомогательные скрипты для локальной установки)

Особенности:

- самый популярный бот для игры в UNO
- продукт активно поддерживается и дорабатывается, последний коммит на github 06.08.2023
- бот доступен в телеграме, также его можно установить локально
- локализован на несколько языков, кроме английского
- удобный графический интерфейс
- в качестве карт Uno используются стикеры
    - есть поддержка карт для страдающих дальтонизмом
- поддерживает несколько режимов игры с разными правилами
    - классический (classic)
    - быстрый (fast)
    - дикий (wild) - есть возможность блефовать

- может вести несколько игр одновременно
    - в том числе в одном чате (баг или фича?), но тогда эти он путает эти игры (баг)
- использует контейнеризацию (docker)

Известные проблемы/баги: https://github.com/jh0ker/mau_mau_bot/issues

    - Неисправленные:
        - графический интерфейс ломается, если иметь 49+ карт
        - при некоторых условиях не работает проверка на возможность хода некоторой картой, полностью ломается игра (возможен краш)
        - если ходить черной картой и не выбрать цвет, дав таймеру истечь, то игра ломается
        - игры не сохраняются после перезагрузки бота
        - не разделяет (путает) несколько игр в одном чате
        - можно снова зайти в игру, в которой ты уже победил (баг или фича?)

    - Исправленные:
        - невозможность блефовать в wild mode
        - заморозка игры при использовании некоторых карт (происходит timeout в боте)
        - бот иногда перестает работать посередине игры

@zeroxisbot (сейчас не работает)
https://github.com/luizeldorado/uno-telegram-bot

Стек: Python, PostgreSQL

Особенности:
- непопулярный бот, нет изменений в репозитории уже два года
- бот в телеграме в настоящий момент не работает, но можно установить локально
- нет графического интерфейса, выбор карты для хода и прочее только через  текстовые команды
- не всегда ведет игру по официальным правилам, есть расхождения
- хостится на heroku
- нет контроля за временем (в другом боте длительность раундов настраивается)
